# OpenChamber Themes

Custom themes for [OpenChamber](https://github.com/anomalyco/openchamber).

## Installation

1. Create the themes directory (if it doesn't exist):
   ```bash
   mkdir -p ~/.config/openchamber/themes
   ```

2. Copy your desired theme(s):
   ```bash
   cp vercel-light.json ~/.config/openchamber/themes/
   ```

3. Reload themes in OpenChamber: **Settings → Theme → Reload themes**

4. Select your theme from the dropdown

## Available Themes

**Colors:**
- Primary: `#62a6ff` (Blue)
- Keywords: `#f05b8d` (Pink)
